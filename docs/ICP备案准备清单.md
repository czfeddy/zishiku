# ICP备案准备清单

更新时间：2026-03-10

适用前提：
- 站点服务器放在中国大陆
- 站点要通过域名对外访问
- 备案类型为网站 ICP 备案，不是 APP 单独备案，也不是 ICP 经营许可证

## 你现在就要准备的材料

### 1. 备案主体信息

个人备案通常需要：
- 备案人姓名
- 身份证正反面
- 本人手机号
- 本人邮箱
- 常住地址

企业备案通常需要：
- 企业营业执照
- 法人姓名、证件信息、手机号、邮箱
- 网站负责人姓名、证件信息、手机号、邮箱
- 企业注册地址和通信地址
- 企业公章

## 2. 域名信息

域名必须同时满足：
- 已注册
- 未过期
- 已完成实名认证
- 域名持有者名称与备案主体名称一致
- 域名后缀和注册服务机构可用于备案

你要自查：
- 域名注册商是哪家
- 域名实名认证主体是谁
- 域名是否刚过户或刚改实名

如果刚改过实名，通常要等注册商把信息同步到工信部系统后再提备案。

## 3. 云服务器与接入信息

你要准备：
- 中国大陆云服务器
- 云厂商备案服务码或等价接入凭证（不同厂商名称不同）
- 服务器所在省市
- 公网 IP

常见情况：
- 服务器在香港、澳门、台湾或海外：通常不走中国大陆 ICP 备案
- 域名已备案但换了云厂商：一般还要做接入备案/新增接入

## 4. 网站本身

备案审核阶段通常需要：
- 网站能打开
- 首页可正常访问
- 网站名称、网页标题、备案提交信息尽量一致
- 网站内容与备案主体、备案备注一致
- 不要放违法违规内容

备案通过后还要补上：
- 首页底部显著位置展示 ICP 备案号
- 如有公安备案要求，再展示公安备案信息

## 5. 可能涉及前置审批的高风险内容

如果网站实际提供以下服务，备案前可能需要额外审批或说明：
- 金融
- 新闻
- 出版
- 医疗、药品、医疗器械
- 文化、广电
- 宗教
- 校外培训
- 网约车等特定行业

如果只是普通企业官网、展示站、知识库，一般按普通网站备案；但页面文案不能做成实际受监管业务。

## 当前项目建议的备案路径

更稳妥的顺序：
1. 先确定备案主体是个人还是公司
2. 确认域名实名和备案主体完全一致
3. 确认服务器在中国大陆并拿到备案接入资源
4. 把站点首页整理成清晰、稳定、合规的展示内容
5. 再在对应云厂商备案系统提交

## 提交备案时你通常要填写的信息

建议你提前整理成表，避免反复退回：
- 备案主体名称
- 主体证件号
- 主体负责人姓名、手机号、邮箱
- 网站负责人姓名、手机号、邮箱
- 域名
- 网站名称
- 服务内容简介
- 服务器 IP
- 接入商
- 应急联系电话

## 你必须自己完成的部分

这些一般必须你本人或企业管理员操作：
- 域名注册/过户/实名认证
- 云厂商实名认证
- 在云厂商备案系统提交主体资料
- 人脸核验或真实性核验
- 手机短信核验
- 上传营业执照、身份证等证件
- 根据管局退回意见补资料
